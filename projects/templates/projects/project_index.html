Inside My Video Hub...

My Video Hub at a glance!

My Video Hub is a free video sharing website that makes it easy to create and watch video playlists online. The website utilizes YouTube's API to access the vast database that is connected to the world's most popular video content website.

My Video Hub is simple, easy to use, and focused on each user sharing video Hubs (e.g. video Playlists) that speak to their specific interests and personality characteristics.

Designed to securely scale and handle rapid user growth, My Video Hub is built with the high level Django web development framework. Django helps to ensure that the web application can handle massive spikes in web traffic at any given time.

Some of the busiest sites on the web leverage Django’s ability to quickly and flexibly scale. Therefore, Django is the ideal web framework to integrate with YouTubes popular API.

The DRY or “Don’t Repeat Yourself” principle is a software development practice aimed at reducing repetition of information.

MVH was built with the DRY design principle in mind, resulting in a simple database designed to hold two classes, one class for the User to create Hubs and another class for a signed in User to create and modify individual videos.

HTML5 is a markup language used for structuring and presenting content on the World Wide Web. HTML design principles were used to structure the site into sections such as Header, Nav, Main, and Footer.

In addition, Bootstrap's grid layout system was implemented via containers and other best practices to ensure a mobile first design was achieved. Bootstrap5 is the newest version of Bootstrap and the most popular CSS Framework for developing responsive and mobile-first websites.

Django REST framework is a powerful and flexible toolkit for building Web APIs. My Video Hub utilizes REST API serializers to fully support both ORM and non-ORM data sources.

Additionally, I implemented a default API root view with format suffix patterns attached to the original URLs to create a user-friendly web browsable API.
